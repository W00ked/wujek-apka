## Replacement guidance

For production, replace these with clean source assets at the same filenames or update the `src` paths in the relevant composition files.

Recommended production dimensions:

- Hero meal photo: at least 2160x1640, JPG/WebP.
- Ingredient thumbnails: 340x340 or 512x512, square, JPG/WebP/PNG.

All image clips in the template already define fixed dimensions and `object-fit: cover`.
